🤖 Telegram Bot Setup

SmartExpense Telegram Bot lets you add expenses instantly using simple commands!

Bot Commands:
/start → Welcome message
/expense food 350 lunch → Add expense
/balance → Check remaining budget
/summary → Weekly breakdown
/insights → AI suggestions
/help → All commands

Step 1: Open the Bot

Telegram pe search karo: @smartexpense_ai_bot  aur /start bhejo.

Step 2: Apna Telegram ID pata karo

⚠️ Username nahi, numeric ID chahiye!
Bot pe /start bhejne ke baad bot tumhara numeric ID reply karega (e.g. 987654321). Woh number yahan paste karo.

Step 3: Link Your Account

✅ How it works:
1. Telegram pe @userinfobot search karo aur /start bhejo — woh tumhara numeric ID batayega (e.g. 7234567890)
2. Woh ID yahan paste karo aur Link karo
3. @smartexpense_ai_bot Bot pe commands use karo expenses add karne ke liye
4. Sab expenses dashboard mein auto-sync honge
